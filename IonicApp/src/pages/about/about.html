About
Die App 'NetzAmpel' dient zur Steuerung eine umgebauten Bobbycar-Ampel via Azure Message Queue. Sie können, nachdem Sie die Daten der zugehörigen Azure Function angegeben haben, die Ampel mit den Buttons steuern.
With the app 'NetzAmpel' You can switch the hacked Bobbycar-Signallight via Azure Message Queue. After filling up the connection info for the Azure Function You can control the signal lights withe buttons on the home tab.
